# 真佛宗術語修正對照表

語音辨識（Speech-to-Text）產生的逐字稿中，佛教術語和專有名詞經常被誤認。以下是常見的錯誤與正確對照。

## 修正原則

1. 根據上下文判斷正確用詞
2. 佛教術語以正式經典用語為準
3. 人名、地名以通用寫法為準
4. 不確定時寧可省略該段，不要猜測

## 稱呼慣例

### 對根本上師的稱呼（報導中使用）
- **聖尊** — 最正式的稱呼，用於報導敘述
- **法王**、**蓮生法王** — 常用稱呼
- **蓮生活佛** — 全稱
- **師尊** — 弟子角度的稱呼，引述弟子的話時使用

### 其他稱呼
- 上師：稱「XX上師」
- 法師：稱「XX法師」  
- 弟子提問：使用法名，如「蓮花XX」
- 師母：蓮香師母（蓮香上師）

## 注意事項

- 這份對照表並非完整列表，僅收錄已確認的常見錯誤
- 遇到不確定的術語時，應搜尋真佛宗官方資料或既有報導來確認
- 有些詞彙在不同上下文中可能有不同的正確寫法，需根據語境判斷
